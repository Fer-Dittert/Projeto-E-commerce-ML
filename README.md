# Projeto de Machine Learning para E-commerce

## 📌 Sobre o Projeto

Este é um projeto de **Machine Learning aplicado a e-commerce**, desenvolvido em grupo com o objetivo de ir além de um trabalho acadêmico tradicional.

A proposta não é apenas construir um modelo de ML, mas sim **simular um ambiente real de trabalho em equipe**, envolvendo:

- Organização de código
- Versionamento com Git
- Desenvolvimento colaborativo
- Estruturação de dados e pipelines
- Boas práticas de engenharia de dados e ML

---

## 🎯 Objetivos

- Desenvolver uma solução de Machine Learning aplicada a dados de e-commerce
- Estruturar um projeto seguindo boas práticas do mercado
- Trabalhar de forma colaborativa, simulando um time de dados real
- Criar um pipeline organizado, desde os dados brutos até a modelagem

---

## 🏗️ Estrutura do Projeto (em evolução)

O projeto seguirá uma arquitetura baseada no conceito de **Medallion Architecture**:

- **bronze** → dados brutos
- **silver** → dados tratados e limpos
- **gold** → dados prontos para análise e modelagem

Além disso, o projeto incluirá:

- Scripts de ingestão e transformação de dados
- Construção de features para Machine Learning
- Desenvolvimento e avaliação de modelos
- Possível integração com ferramentas de visualização
